# Clone Disney plus

A clone of the Disney Plus page developed with bootstrap

## Quickstart

1. Clone this project

   ```shell
   git clone https://github.com/KauaOnoratoDev/clone-disneyplus.git
   ```

2. Install dependencies:

   ```shell
   cd clone-disneyplus
   npm install
   ```

3. Run local dev server:

   ```shell
   npm run build
   start index.html
   ```
