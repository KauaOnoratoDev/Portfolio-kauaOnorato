# Calculadora IMC

A BMI calculator powered by Vite

## Quickstart

1. Clone this project

   ```shell
   git clone https://github.com/KauaOnoratoDev/Imc-React.git
   ```

2. Install dependencies:

   ```shell
    cd Imc-React
   npm install
   ```

3. Run local dev server:

   ```shell
   npm run dev
   ```
